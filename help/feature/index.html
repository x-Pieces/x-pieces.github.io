如何自定义功能项？

1 分钟

小零件中内置了大量常用功能以及热门App的启动功能，若无法满足您的需求，您可以自定义功能项。在功能页面：

  • 点击查看推荐使用功能可查看内置的大量常用功能。
  • 点击右上角+可添加自定义功能项，通过设置链接来实现自定义功能。

链接设置 #

打开任意网页 #

在网址输入框输入任意您想快速访问的网站地址，测试后保存即可，使用了此功能的小组件点击后将会跳转浏览器并打开指定的网站。

调用快捷指令 #

快捷指令为iOS系统自带的强大App，可为您喜爱的应用创建包括多个步骤的个性化捷径指令，您在快捷指令设定好的自动化功能， 均可以应用到小组件上一键调用，利用此特性，可以让小组件的功能更加强大， 包括 打开任意App 等等。

设置步骤如下：

  1. 打开快捷指令，添加一条指令，并命名为X（X为可以为任意字符，例如"打开微信"、“播放音乐"等等）
  2. 然后，在指令名称输入框中输入名称X，注意必须与在快捷指令App中设置的名称完全一致才能正确调用。
  3. 测试后保存即可。

自定义 #

URL Scheme是App提供给外部直接操作App的规则，利用它可以快捷唤起相关App并执行相关操作， 例如 打开任意微信小程序、打开任意支付宝小程序 等等。

在URL Scheme输入框中填写相关应用的URL scheme，保存即可。
